# Azure Storage Account Project
## Project Summary

This project demonstrates the creation and management of Azure cloud resources using Microsoft Azure Portal.

In this lab, I:
- Created an Azure subscription
- Created a Resource Group
- Created an Azure Storage Account
- Uploaded and edited files in Azure Storage
- Downloaded and verified file changes
- Deleted resources to avoid cloud costs

This project helped me gain hands-on experience with cloud infrastructure and Azure resource management.
## Technologies Used

- Microsoft Azure
- Azure Storage Accounts
- Azure Resource Groups
- Azure Portal
- Text Files
## Demonstration

### Step 1
Created an Azure Resource Group.

### Step 2
Created an Azure Storage Account inside the Resource Group.

### Step 3
Uploaded a text file into Azure Storage.

### Step 4
Edited the file directly inside Azure Portal.

### Step 5
Downloaded the file and verified changes.

### Step 6
Deleted the Resource Group to prevent charges.
